# Vendor Contract & Invoice Management System — Frontend

React + Vite + Tailwind CSS frontend for the **Vendor Contract & Invoice Management System (VCIMS)**.

See the [main README](../README.md) for full setup instructions, test credentials, and API documentation.

## Quick Start

```bash
npm install
npm run dev
```

Runs at [http://localhost:5173](http://localhost:5173) with API proxy to `http://localhost:5000`.
